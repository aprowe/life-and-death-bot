# Life and Death Bot

[Riddles.io API](https://docs.riddles.io/game-of-life-and-death/api)


## To install
This project recommends using `pipenv`

In the project folder, simply run
```
pipenv install
```


## To Start
```
python3 src/main.py
```

## Testing
```
python3 test/run_tests.py
```
